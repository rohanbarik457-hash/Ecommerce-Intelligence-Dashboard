# 📊 E-Commerce Product Intelligence Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard developed to analyze e-commerce product data. It provides insights into product performance, pricing, discounts, customer ratings, revenue potential, and stock availability to support better business decisions.

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## 📈 Features
- Product Performance Analysis
- Category & Brand Analysis
- Revenue Potential Insights
- Customer Ratings & Reviews
- Discount Analysis
- Stock Availability Tracking
- Interactive Filters & Visualizations
